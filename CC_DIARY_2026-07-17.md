## 踩过的坑

1. **不偷偷降级** — graph-rag-agent 缺依赖→报错，不能自动切 SimpleGraph。这是红线。
2. **Qwen 返回 prompt 原文** — 必须用 chat_template，不能裸调 generate
3. **IdentityWeight 前缀嵌入** — 维度不匹配(256 vs 1536)，直接拼接会破坏推理。改为 system message
4. **Windows GBK 编码** — bat 文件中文乱码，改为纯 ASCII
5. **accelerate 未安装** — device_map="auto" 失败，降级手动 cuda()
6. **CUDA vs CPU** — Hermes venv 无 CUDA，系统 Python312 有。模型用系统 Python 加载
7. **硬编码陷阱** — 多次被张凯纠正，不能"看上去没问题"。prompt 要从文件读取
