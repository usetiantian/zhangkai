## 七、踩过的坑

1. **幂等性**: 同一个操作今天重复了多次 (git init 多次重建)。需要每次前先检查状态。
2. **路径转义**: Bash 吃反斜杠, PowerShell 变量替换也不可靠 → 用 .ps1 文件调用最稳。
3. **Unicode/Emoji**: Windows 控制台 GBK 编码炸了多次 → 用 `PYTHONIOENCODING=utf-8` 或用 ASCII 替代。
4. **嵌套 git**: 工作区有多个子仓库 (grok-build/ECC/laap-agi/uzi-skill) → .gitignore 排除。
5. **Windows 保留文件名**: `nul` 文件导致 git add 失败 → .gitignore 排除。
6. **GitHub Push Protection**: 飞书密钥被拦截 → 从历史中移除, 文件加入 .gitignore。
7. **adata SSL**: Baidu finance API SSL 错误 → pytdx 是稳定的主力。
8. **Qwen 角色扮演**: LM Studio 未运行时降级为简易规则投票, 系统不中断。
