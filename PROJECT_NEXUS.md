## 一、产品定义

### 一句话

**每个人的专属 AI，越用越懂他，能满足他的任何需求。**

### 拆解

| 关键词 | 含义 | 竞品对比 |
|--------|------|------|
| **专属** | 私有化部署，数据不出家门 | ChatGPT/Claude 数据上云端 |
| **越用越懂** | 三层记忆持续学习，不分人 | Siri 不记得上次聊什么 |
| **任何需求** | 技能包生态，社区贡献扩展 | 所有竞品功能固定不可扩展 |

### 差异化定位

| | ChatGPT/Claude | Siri/小爱 | **NexusAgent** |
|------|:--:|:--:|:--:|
| 归属 | 公司 | 公司 | **你的** |
| 数据 | 云端 | 云端 | **本地（不出家门）** |
| 记忆 | 遗忘 | 遗忘 | **三层记忆，越用越懂** |
| 能力 | 固定 | 固定 | **技能商店，无限扩展** |
| 付费 | 月租 | 免费 | **买断制** |

## 二、架构设计

### 部署架构

```
┌──────────┐      WiFi       ┌──────────────────┐
│ 手机/电脑  │◄──────────────►│ PC 或 Nexus Box   │
│ 只做终端  │  语音输入/输出  │                  │
│ 屏幕+麦   │               │  AI 大脑          │
│          │               │  记忆/推理/成长     │
└──────────┘               │  数据不出家门      │
                           └──────────────────┘
```

### 两阶段路线

| 阶段 | 产品形态 | 硬件要求 | 目标用户 |
|------|---------|---------|---------|
| 第一阶段 | PC 软件版 | 用户自有电脑 + 中端显卡 | 极客/开发者/重度 AI 用户 |
| 第二阶段 | Nexus Box | 专用硬件盒子（类似 Mac Mini） | 普通消费者 |

### 核心原则

- 一个软件，两种部署：PC 版和 Box 版跑同一套代码
- 手机不装 AI，手机只是遥控器
- 数据物理级隐私——永远不出用户家门

## 三、平台能力

### 底层能力（平台提供，对所有人一样）

| 能力 | 说明 | 技术路线 |
|------|------|------|
| 推理引擎 | CC 级分析和决策 | Claude API（PC版）/ 本地模型（Box版） |
| 记忆系统 | 三层记忆，越用越懂 | 短期(对话)→中期(MCP知识图谱)→长期(Git) |
| 语音交互 | 说话就行 | Whisper(听) + 本地TTS(说) |
| 隐私加密 | 数据不出家门 | 本地存储 + 加密传输 |
| 成长机制 | 用越久越聪明 | Constitution约束下的进化 |
| 可继承 | 数字遗产 | 加密导出/导入 |
| 一键安装 | 普通人也能装 | exe打包 |

### 上层技能（社区贡献，无限扩展）

```
NexusAgent 平台
├── CAD 画图技能    ← "画承重墙4米的户型图" → 出图
├── 法律咨询技能    ← "类似案子判例" → 案例
├── 医疗辅助技能    ← "鉴别诊断" → 分析
├── 股票分析技能    ← 乾坤·UZI（已跑通，第一个示范技能）
├── 编程导师技能    ← "教我这个算法"
├── 会计报税技能    ← "算进销项差额"
├── ... 任何职业 x AI
```

**平台不做技能内容。只提供"听懂→理解→调度→输出"的底层通道。**

## 四、商业模式

### 收入来源

| 来源 | 定价 | 说明 |
|------|------|------|
| Box 硬件 | ¥3000-5000 | 一锤子买卖，硬件成本+毛利 |
| 技能商店抽成 | 30% | 开发者上架技能包，用户付费下载 |
| 企业版 | 按需报价 | 全员部署+私有化定制 |
| PC 软件版 | 免费/低价 | 培育用户基盘 |

### 关键对标

- 苹果：iOS + App Store → 硬件+抽成
- NexusAgent：Box + 技能商店 → 硬件+抽成

## 五、MVP 定义

> MVP（Minimum Viable Product）= 能拿出去给人试的最简版本

### MVP 包含

| 功能 | 状态 |
|------|:--:|
| 记忆系统（三层） | ✅ 已跑通 |
| 推理引擎（CC级） | ✅ 已跑通 |
| 手机连接（app.html） | ✅ 已跑通 |
| 语音交互（说话就行） | ⚠️ 待接入 Whisper + TTS |
| 一键安装（exe打包） | ⚠️ 待开发 |
| 第一个技能（乾坤·UZI 股票） | ✅ 已跑通 |

### MVP 体验

1. 用户下载安装 → 手机扫码连接
2. 对着手机说话："今天有什么超短线机会"
3. AI 扫描全市场 → 语音回复结果
4. 关了再开，AI 记得上次聊的
