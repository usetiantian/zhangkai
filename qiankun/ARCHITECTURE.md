# 乾坤 (QianKun) — A 股 AI 研究助手

## 定位

7×24 小时自主运行的 A 股智能研究员。
主动扫描全市场 → 发现机会 → 深度研究 → 推送结论。
同时支持问答交互。

## 四层架构

```
┌─────────────────────────────────────────────────┐
│                 入口层 (Gateway)                  │
│  CLI / 飞书 / Web Dashboard / 定时调度           │
└──────────────────────┬──────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────┐
│                 认知层 (Brain)                    │
│  意图理解 → 任务分解 → 研究执行 → 结论生成       │
│  市场扫描 / 深度研究 / 复盘总结 / 异动响应        │
└──────────────────────┬──────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────┐
│                 能力层 (Strategy)                 │
│  技术分析 / 基本面分析 / 消息面分析 / 策略回测    │
│  选股模型 / 风险评估 / 研报解读 / 情绪分析        │
└──────────────────────┬──────────────────────────┘
                       │
┌──────────────────────▼──────────────────────────┐
│                 数据层 (Data)                     │
│  akshare(主) / tushare(备) / baostock / SQLite  │
│  行情 / 财报 / 公告 / 新闻 / 龙虎榜 / 北向资金   │
└─────────────────────────────────────────────────┘
```

## 工作节奏

```
09:15  开盘前扫描 → 推送今日关注
09:30  开盘盯盘 → 异动实时监控
11:30  午间快报
15:00  收盘复盘 → 策略扫描 → 推送选股结果
17:00  深度研究 → 财报分析 / 研报解读
20:00  每日研报 → 飞书推送
周末    周度复盘 + 策略优化
```

## 复用 Nexus 的代码

| 来源文件 | 用途 |
|------|------|
| `nexus_llm.py` | DeepSeek API 客户端（同步/异步/流式） |
| `error_classifier.py` | 统一错误处理 |
| `event_bus.py` 模式 | 模块间发布订阅 |
| `checkpoint.py` 模式 | 状态持久化 |
